## Introdução

O Dashboard de Processos é uma aplicação web que permite visualizar e analisar informações sobre os processos em execução em diferentes computadores. Ele oferece uma interface amigável e gráficos informativos para facilitar a interpretação dos dados.

## Funcionalidades

- **Visualização de Processos:** Lista e analisa os processos em execução em computadores específicos.
- **Filtragem e Seleção:** Permite filtrar por andar, selecionar processos para análise detalhada e excluir processos específicos.
- **Relatórios com Gráficos:** Gera relatórios com gráficos informativos, incluindo quantidade de processos, uso de CPU e RAM.
- **Recursos Adicionais:**
    - **Copiar ID do Computador:** Facilita o uso do ID único do computador.
    - **Atualização Dinâmica:** Gráficos atualizados em tempo real para uma análise em constante evolução.

## Instalação

1. Clone o repositório para sua máquina local:

   ```bash
   git clone https://github.com/Centrix-Solutions-Grupo-07/Projeto_Individual_Alexandre.git
   ```

2. Abra o arquivo `index.html` em um navegador web.

## Como Contribuir
